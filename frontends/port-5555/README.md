# Port 5555 - Main React Frontend

## Description
Primary Pokemon TCG Search application built with React 18 + TypeScript.

## Features
- Modern React with TypeScript
- Tailwind CSS v4 styling
- Pokemon TCG API integration
- Grid and List view modes
- Detailed card modal displays

## Tech Stack
- React 18
- TypeScript
- Vite 7
- Tailwind CSS v4
- Axios for API calls

## Running
```bash
npm run frontend:5555
```

Access at: http://localhost:5555

## Use Case
Primary production frontend with full features and modern React architecture.
